# Angular ToDoApp - simple 'todo app' created in Angular 2X with:

- angular components
- angular modules
- angular router
- scss
- REST Api (json-server)
- Simply token authentication

# Install and run

## Install npm modules
```bash
cd todo-app
npm install
```

## Run project
```bash
ng serve
```

## Run json-server
```bash
node json-server.js
```
